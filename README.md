Prerequisite:
=============
1) PlasticSCM must download and its version should be 11.0.16.+ but lower than 2021.
2) PlasticSCM should login.
3) DTH_Content_Workflow project should download.
4) DTH_Content_Workflow editor should close.

Now Open App:
=============
1) Select Organization.
2) Select Unity version.
3) Enter name for new project.
4) Select project location for new project.
5) Now, click create button.

What is working in background:
==============================
1) Checking plasticscm is login or not.
2) Checking DTH_Content_Workflow project download or not.
3) Creating folder for new project.
4) Check pending changes available or not.
5) If pending changes there Undo option will popup. if "yes" all changes will undo if "no" this will skip.
6) Switch to main branch or Unity 6 branch.
7) Checking DTH_Content_Workflow changeset.
8) Creating new repository for new project.
9) Now, all DTH_Content_Workflow work will push in new repository.

Note: The branch name and project name may change from time to time, but the Plastic version will change only in rare cases.
